## 展開運算子(spread operator)/其餘運算子(rest operator)

**展開運算子**    
  
    const arr = [1,2,3,4];
    
    const arr1 = [0, ...arr];
    
    console.log(arr1);
  
**其餘運算子**  
  
    const arr = [1,2,3,4];
    
    const [a, ...arr1] = arr;
    
    console.log(arr1); // 1
    
    console.log(arr1); //[2,3,4]
